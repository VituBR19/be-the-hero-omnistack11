## Be The Hero Omnistack 11

Projeto feito com o acompanhamento da **Semana Omnistack** gerenciada pela equipe da [Rockseat](https://rocketseat.com.br/).

  A ideia do projeto é simples, uma ONG precisa de apoio financeiro para poder tratar de alguns casos,
então a plataforma permite com que uma ONG faça seu login e cadastre os casos que precisam de apoio
financeiro, para que um cidadão de bem possa contribuir, doando a quantia necessária para ajudar no caso.

## Como instalar
Para instalar essa aplicação basta:
- Clonar o repositório.
- Na raiz do repositório execute o comando `npm install` no terminal

## Tecnologias
Para a construção do app, foi usado:
- Nodejs para a criação do backend.
- Reactjs para a criação do app na web
- React Native para a criação do app mobile
